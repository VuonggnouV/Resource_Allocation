# Hướng dẫn chụp ảnh minh họa (API Screenshots)

Để hoàn thiện sản phẩm bàn giao số 5 (API Screenshot) theo đúng yêu cầu đề bài:

1. Đảm bảo docker-compose đã được khởi động và chạy thành công (`docker-compose up --build -d`).
2. Mở trình duyệt và chụp lại 3 màn hình sau:
   - **Tấm 1**: Trang **Swagger UI** tại địa chỉ: `http://localhost:8080/swagger-ui/index.html` (Đặt tên file: `api_swagger_ui.png`).
   - **Tấm 2**: Trang **Dashboard** chính của ứng dụng tại: `http://localhost/dashboard` (Đặt tên file: `ams_dashboard.png`).
   - **Tấm 3**: Trang danh sách **Nhân Sự** tại: `http://localhost/employees` (Đặt tên file: `ams_employees.png`).
3. Lưu đè cả 3 file ảnh này vào đúng thư mục `screenshots/` này là xong!
